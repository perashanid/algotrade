# Landing Page Components

Modern, animated landing page with smooth transitions and elegant effects.

## Features

- **Smooth Animations**: Framer Motion powered animations throughout
- **Scroll Reveal**: Elements fade and slide into view as you scroll
- **Typewriter Effect**: Animated text that appears word by word
- **Responsive Design**: Mobile-first design that works on all devices
- **Dark Mode Support**: Full dark mode compatibility
- **Modern UX**: Parallax effects, gradients, shadows, and smooth transitions

## Components

### Sections
- **Navbar**: Sticky navigation with smooth scroll and mobile menu
- **Hero**: Eye-catching hero section with animated elements
- **Features**: Grid of feature cards with hover effects
- **HowItWorks**: Step-by-step process visualization
- **Testimonials**: Customer testimonials with ratings
- **Pricing**: Pricing plans with feature comparison
- **CTA**: Call-to-action section with gradient background
- **Footer**: Footer with links and social media

### Utilities
- **AnimatedText**: Word-by-word text animation component
- **ScrollReveal**: Scroll-triggered animation wrapper

## Usage

The landing page is automatically shown at the root path `/`. Users are redirected to `/app/dashboard` after login.

## Color Scheme

The design uses a consistent color palette defined in `tailwind.config.js`:
- Primary: Blue shades (#3b82f6)
- Success: Green shades (#22c55e)
- Danger: Red shades (#ef4444)
- Gradients: Primary to Indigo, and various feature-specific gradients

## Icons

Uses `react-icons/fi` (Feather Icons) throughout for a consistent, modern look.
